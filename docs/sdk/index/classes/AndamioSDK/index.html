AndamioSDK

Main class for the Andamio SDK. Provides access to the UTXO-RPC client and provider.

AndamioSDK

Example​

const sdk = new AndamioSDK('https://preprod.utxorpc-v0.demeter.run:443', 'Preprod', 'dmtr_utxorpc...');
